# Claude Night/Day

A warm, comfortable VS Code theme inspired by [Claude AI's](https://claude.ai) web interface.

## Themes

### Claude Day
Warm cream background with high-contrast dark text. Easy on the eyes without the harshness of pure white.

### Claude Night
Soft charcoal background with cream text. Comfortable for extended coding sessions.

Both themes use Claude's signature **terra cotta accent** for selections, cursors, and highlights.

## Install

### From GitHub

1. Clone or download this repository
2. Copy to your VS Code extensions folder:
   - **macOS**: `~/.vscode/extensions/claude-night-day-theme`
   - **Windows**: `%USERPROFILE%\.vscode\extensions\claude-night-day-theme`
   - **Linux**: `~/.vscode/extensions/claude-night-day-theme`
3. Restart VS Code

### From VSIX

1. Download the `.vsix` from [Releases](https://github.com/tkwn2080/claude_vscode/releases)
2. In VS Code: Extensions > `...` > Install from VSIX

## Activate

`Cmd+Shift+P` (Mac) or `Ctrl+Shift+P` (Windows/Linux)

Type "Color Theme" and select **Claude Day** or **Claude Night**
