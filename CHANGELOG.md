# Changelog

All notable changes to the "Claude Night/Day" theme will be documented in this file.

## [1.0.0] - 2024-11-25

### Added
- Initial release
- **Claude Day** - Warm cream light theme
  - Pampas (#F4F3EE) editor background
  - Dark brown (#3d3929) text for high contrast
  - Terra cotta (#da7756) accents
- **Claude Night** - Comfortable dark theme
  - Warm charcoal (#201f1d) editor background
  - Cream (#e8e6de) text for reduced eye strain
  - Terra cotta (#da7756) accents
- Comprehensive workbench color coverage
- Syntax highlighting for all major languages
- Semantic highlighting support
- Terminal color scheme matching theme palette
